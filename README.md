# theseed-skin-buma

This is a third-party skin for the seed engine, based on [bulma](https://bulma.io).
Special thanks to [gdl-888](https://github.com/gdl-888) for porting to Vue.js.

## Build
You need [Yarn](https://yarnpkg.com/) to build this skin.
```bash
yarn
yarn build
```

## Formatting
